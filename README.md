# 🛢️ CRUDE EMPIRE — Oil & Gas Tycoon (3D Edition)

A fully immersive oil & gas upstream simulation game with a 3D map, animated drilling rigs, cinematic main menu, and realistic well economics — all in a single HTML file.

## 🎮 [Play Now →](https://ibambekov1.github.io/crude-empire/)

## Features

- **3D Interactive Map** — Click basins on the Three.js-powered terrain to acquire leases
- **Cinematic Main Menu** — Animated particle network background
- **5 Major Basins** — Permian, Bakken, Eagle Ford, DJ Basin, SCOOP/STACK
- **Animated Drilling Rigs** — Watch rigs drill and pumpjacks produce in real-time 3D
- **Realistic Well Economics** — Arps hyperbolic decline, rising GOR & water cut
- **Artificial Lift** — Rod Pump, ESP, Gas Lift, Plunger Lift
- **Random Events** — Blowouts, freeze-offs, frac hits, regulatory fines
- **Production Gauges** — Animated dashboard with charts
- **$100M Starting Capital** — Build your empire

## Quick Start

Just open `index.html` in any modern browser. No server needed.

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to **main branch**, folder **/ (root)**
4. Your game is live at `https://yourusername.github.io/crude-empire/`

## Tech Stack

- **Three.js** r128 — 3D rendering
- **Vanilla JS** — Game engine
- **CSS3** — Animations & effects
- Zero dependencies. Single file. No build step.
